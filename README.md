.Постановка задачи
Разработать программу "Кадровое агентство" для управления базой данных соискателей с возможностью формирования трех видов отчетов с использованием шейкерной сортировки.
Исходные данные:
•	Файл с записями соискателей
•	Каждая запись содержит: фамилия, имя, отчество, пол, дата рождения (год, месяц, число), специальность, стаж работы, знание иностранных языков, ожидаемый оклад
Требуемые функции:
1.	Чтение данных из файла и сохранение в файл
2.	Формирование трех отчетов с использованием шейкерной сортировки:
Отчет 1: Полный список всех соискателей, отсортированный по специальности (по возрастанию) + фамилия (по возрастанию)
Отчет 2: Список всех соискателей заданной специальности, отсортированный по стаж работы (по убыванию) + пол (по убыванию) + фамилия (по возрастанию)
Отчет 3: Список всех соискателей с окладом в диапазоне от N1 до N2 рублей, отсортированный по ожидаемый оклад (по убыванию) + фамилия (по возрастанию)
3.	Добавление, удаление, редактирование и просмотр записей
4.	Защита от некорректного ввода данных
5.	Удобный пользовательский интерфейс
Технические требования:
•	Использование алгоритма шейкерной сортировки
•	База данных должна содержать не менее 25 записей
•	Программа должна корректно работать с различными исходными данными
